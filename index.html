HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A child element is contained within a parent element.
  • A tag is the beginning and end of an element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style with CSS. Inline CSS, Internal CSS, and External CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Here are features in JavaScript that are common to all coding languages. Master these fundamentals:

    1. Variables - Containers that store values.
    2. Comments - Snippets of text that can be added along with code.
    3. Operators - Mathematical symbol that produces a result based on two values or variables.
    4. Conditionals - Code structures used to test if an expression returns true or not.
    5. Functions - A way of packaging functionality that you wish to reuse.
    6. Events - Real interactivity on a website requires event handlers. These are code structures that listen for activity in the browser, and run code in response. The most obvious example is handling the click event, which is fired by the browser when you click on something with your mouse.

  • Primitive Values:
  • Null, Undefined, Boolean, Number, Big Int, String, Symbol.

  • Symbols can be used as a key part of an object property.
  • Objects - A value in memory which is possibly referenced by an identifier. In JavaScript they are only mutable values. Functions are in fact, also objects with the additional capability of being callable.
  • Properties - In JavaScript objects can be seen as a collection of properties.

  • Control Flow - The order in which a computer executes code in a script which is always top to bottom.
